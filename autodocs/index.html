Expr functions

Below are functions which can be used in the expression evaluation node, expr.

name params opt. params description
clip image clip all channels of an image to 0-1
curve image,mul,add impose a sigmoid curve on an image, y=1/(1+e^-(mx+a))) where m and a are parameters
norm image splitchans normalize all channels of an image to 0-1, operating on all channels combined (the default) or separately
